# @usfm-tools/editor-ui

WYSIWYG chrome for the USFM ProseMirror editor: block gutter (+), marker palette, selection bubble, and sync conflict review UI.

Import **`@usfm-tools/editor-ui/chrome-ui.css`** alongside **`@usfm-tools/editor/chrome.css`**.

See [`docs/25-editor-ui-toolkit.md`](../../docs/25-editor-ui-toolkit.md) in the monorepo.
